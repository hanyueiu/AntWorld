# Python

- ~PyQt5相关安装

  ```shell
  pip install PyQt5 -i https://pypi.douban.com/simple
  # 出现unpack error时依然可以安装 
  pip install -i https://pypi.douban.com/simple --trusted-host pypi.douban.com pyqt5-tools
  # Anaconda下安装两个包成功后, 应该就可以配置下面两个命令, 成功运行
  pip install -i https://pypi.douban.com/simple --trusted-host pypi.douban.com pyqtwebengine
  pip install -i https://pypi.douban.com/simple --trusted-host pypi.douban.com pyqtchart
  pip install -i https://pypi.douban.com/simple --trusted-host pypi.douban.com qscintilla
  ```
